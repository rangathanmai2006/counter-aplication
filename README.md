# Counter Application

A simple and interactive Counter Application built using HTML, CSS, and JavaScript.

## 🚀 Features

- ➕ Increase Counter
- ➖ Decrease Counter
- 🔄 Reset Counter
- 📱 Responsive Design
- 🎨 Simple and Clean UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
counter-application/
│── index.html
│── style.css
└── script.js
```
